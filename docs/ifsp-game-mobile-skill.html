IFSP Game Mobile UI

Corrida das Expressões

Direção: jogo educativo mobile-first com base verde IFSP, botões grandes, carrinho/avatar central, pista 2D e feedback positivo.

Paleta IF

Verde #2f9e41, vermelho #cd191e, preto #000000, branco #ffffff. Open Sans como fonte principal.

Paleta game

Amarelo para ação, azul/roxo/laranja/rosa/ciano apenas para personalização e energia visual.

Componentes

Pista 2D

Regras de marca

  • Não recriar logo IF manualmente como marca oficial.
  • Não distorcer, rotacionar, alterar cores ou aplicar efeitos na marca.
  • Não usar a marca como marca d'água; módulos quadrados podem ser inspiração abstrata.
  • Em telas de jogo, manter verde IF como base e cores vivas apenas em elementos de jogo.
